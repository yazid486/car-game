# VINTAGE RACER: AEGIS-X PROTOCOL

![Vintage Racer Hero](https://images.unsplash.com/photo-1511919884226-fd3cad34687c?auto=format&fit=crop&q=80&w=1000)

**Vintage Racer** is a high-fidelity, technical racing game built with vanilla web technologies. Designed with a sleek, industrial "Aegis-9" aesthetic, it features deep vehicle customization, dynamic 2-player local multiplayer, and a responsive tactical interface.

## 🚀 Features

- **Technical Aesthetic**: A premium "Aegis-9" design language featuring orbital halo modules, technical geometric markers, and vibrant industrial color palettes.
- **Dual-Pilot Mode**: Local 2-player racing support with independent customization.
- **Deep Customization**: 
    - **Chassis Modules**: Select from a fleet of high-performance frames including the *Hot Rod*, *Striker-X (PX)*, *Heavy Muscle*, *Phantom Specter ✦*, and *Cyber Apex*.
    - **Color Cores**: Fully customizable Primary and Secondary color systems including high-fidelity gradients (Heat, Cool, Nebula, Hazard).
- **Tactical HUD**: Real-time Heads-Up Display tracking "Core Sync" (score) and "Module Velocity" (speed).
- **Advanced Thruster Systems**: Dynamic, vehicle-specific plasma and exhaust thruster arrays that react in real-time to acceleration.
- **Fluid Mechanics**: Dynamic steering tilt and high-speed motion trails powered by Anime.js.

## 🏎️ Vehicle Fleet

Each chassis module features unique geometric architecture and visual details:
- **Hot Rod**: A classic custom beast completely redesigned with massive drag slicks, exposed front suspension, flame decals, and a highly detailed supercharged V8 engine.
- **Striker-X (PX)**: High-speed pursuit module featuring aggressive winglets, advanced sensor arrays, and wing-mounted weapon/thruster pods.
- **Heavy Muscle**: An American powerhouse with a long hood, racing stripes, and chrome bumpers.
- **Phantom Specter ✦**: A stealth-oriented sci-fi fighter chassis with forward-swept horns, blaster cannons, plasma heat vents, and double winglet stabilizers.
- **Cyber Apex**: The pinnacle of futuristic racing technology. Features advanced aero vanes, glowing intakes, a central spine, safety halo, and massive rear diffusers.

## 🎮 Controls

The game is optimized for keyboard interaction.

### Player 1 (MODULE_P1)
- **Accelerate**: `W`
- **Brake/Reverse**: `S`
- **Steer Left**: `A`
- **Steer Right**: `D`

### Player 2 (MODULE_P2)
- **Accelerate**: `↑ (Arrow Up)`
- **Brake/Reverse**: `↓ (Arrow Down)`
- **Steer Left**: `← (Arrow Left)`
- **Steer Right**: `→ (Arrow Right)`

### Global Commands
- **Pause Mission**: `ESC` or the On-Screen Pause Button.
- **Interact**: Use your mouse to navigate the Command Center and Customization Modules.

## 🛠️ Technology Stack

- **Engine**: Vanilla JavaScript utilizing the HTML5 Canvas API for high-performance rendering.
- **Styling**: Modern CSS with Glassmorphism, CSS Variables, and CSS Grid/Flexbox layouts.
- **Animations**: [Anime.js](https://animejs.com/) for smooth, non-linear UI transitions and physical steering tilt effects.
- **Typography**: [Outfit](https://fonts.google.com/specimen/Outfit) via Google Fonts for a readable, geometric, and technical feel.
